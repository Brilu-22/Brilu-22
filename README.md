### Dulce - Mental Wellness Application

<img src="./images/Dulce.png" width="80%" alt="Dulce App Project" />

**Term 3 Project | Mobile Application Development**

**Project Overview:**
An innovative mental wellness application designed to reduce stress through gesture-based interactions and a minimalist, sensory-friendly interface.

**Key Features:**

- Gesture-controlled meditation exercises
- One-color aesthetic (Olive Green #4E6813) for reduced sensory load
- Progressive breathing pattern guides
- Mood tracking with visual analytics
- Accessibility-focused design for neurodiverse users

**Technical Implementation:**

- **Framework:** React Native with Expo for cross-platform compatibility
- **Backend:** Firebase for real-time data synchronization
- **Gestures:** Custom gesture recognition algorithms
- **State Management:** Context API with useReducer

**Academic Focus:** Human-computer interaction, accessibility design, and mobile development best practices.

**Learning Outcomes:** Developed expertise in mobile UX patterns, gesture-based interfaces, and inclusive design methodologies.

**Tech Stack:** `React Native` `Expo` `Firebase` `TypeScript` `Gesture Handling` `Context API`
